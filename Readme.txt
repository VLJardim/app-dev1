Projekt idé
    -Outfit Roulette er en mobilapp, der genererer dagens outfit baseret på vejret, temperatur og farveharmoni. 
    -Brugeren uploader sit eget tøj og får AI-styrede forslag til, hvordan det kan kombineres på nye og stilfulde måder. 
    -Målgruppen er unge, modebevidste brugere, der vil spare tid om morgenen og samtidig lære at udnytte deres garderobe bedre. 
    -Appen er udviklet som mobilapp, fordi den bruger kamera og GPS til at registrere tøj og vejrinformation 
     (fordi outfit-valg sker spontant i hverdagen, hvor mobilen altid er ved hånden).

Features
    -kamera
    -weather forecast
    -AI
    -save outfits
    -input tøj via kamera
    -"closet"
